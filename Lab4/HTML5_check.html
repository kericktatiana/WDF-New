Eurostile

This is a campaign site for a font called Eurostile. Though this design is utilizing the advantages of HTML5, there are still a couple of mistakes lurking around.

  • The HTML document itself is a bit messy and hard to read
  • The first couple of sections are names, the rest remain to be untitled, to fix this you would go into your sections, divs, articles, etc. to add titles.

Conecta Balear

This is an Europe based sites for connecting computers through wifi. The designer of this page knew about the HTML5 structure so the only thing wrong with the outline is that the navigation is being considered an untitled section, to fix that you would just assign a class name to the navigation tag.


Mouse Will Play

This is a portfolio site of a web designer showing off the sites he created and the services he can offer. This designer's code unlike the pervious sites is really clean and easy to read. Unfortunately most of the sections are untitled, the simple fix for that is to go into the code and assign titles.


Bad Bug T-shirts

This last sites is an online store for some awesome looking shirts with amazing clean code, and some comments left for the designer to follow (I'm a little jealous). with only one untitled section which can easily be fixed.
